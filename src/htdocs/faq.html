The Answers (and also the questions again)

 

What is Pasty?

Pasty is a collection of applications that, in total, provide a cloud-based clipboard service to help you easily share short snippets over your different devices. And it's completely free! Take a look at About Pasty for more infos.

 

Why is it free?

It is free because I like services that are free and hence try to offer all my services for free. Of course I can only do this as long as I can afford it. What was that? You wanna help me finance this? Well, isn't that kind of you! Check out the Donations Page.

 

What about my data, do you make money using my data?

Nope. I don't like services that make money using their users data. Just because I offer you a service to transfer data for you does not mean I have any rights to that data. It's yours, and it stays that way. The only way I can make money and hence pay for this service is via donations. So if you like Pasty, consider donating a buck or two.

 

What clients are there?

Currently there is a native Android app, a rather crude Web App, and a GTK client is in work. What I wanna build next is a Mobile Webapp, an Ubuntu client, and a Windows client.

 

When will client xyz be released?

Considering I do all this in my free time, development can be slow. I always try my best to deliver the clients I deem most important asap. But Pasty's API is open, so if you are somebody or know somebody that knows how to code, check out « I am a developer and want a client for xyz.» .

 

I am a developer and want a client for xyz.

First of all, that's not a question. Ha, I know, I wrote it myself and now act as if it is your fault. Isn't that funny? But enough of teh funnehs, let's get to business. Pasty's API is open, so if you REALLY want that client, why don't you go ahead and write it yourself? Have a look at the Wiki pages on Github, mostly the REST API.

 

Is Pasty Open Source?

Right now, no it isn't. But that is not an issue of closed source vs. open source. I am an 100% open source guy. So Pasty or at least most parts of it being released publically is only a matter of time. You will then find the source on Github. A small part of Pasty is already released as OSS. It is a node.js (Javascript) library that handles API calls to Pasty's API Server. It is used as part of the Web App it is released under the name pastyclient-node and is available via Github and npm. It is licensed under Apache License, Version 2.0.
